Woo Enquiry


== Description ==

woocommerce Product enquiry form whatsapp, call for price and contact form Product Quote Woocmmerce Plugin.

== Installation ==
1. Download and extract the zip archive
2. Upload `woo-enquire` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Click on Woo Enquire From Side mneu
